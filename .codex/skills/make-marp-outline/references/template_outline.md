# Outline（構成案テンプレート）

このテンプレートを使用して、プレゼンテーションの構成案を作成します。

---

## 時間配分（例: 10分）

1. 導入（0:45）
2. 問題提起（2:00）
3. 転換点（1:30）
4. 解決策（2:30）
5. 具体例（2:00）
6. 展望（1:00）
7. クロージング（0:15）

---

## 各セクションの狙い

- **導入**: 聴衆の注意を引く。共感ポイントを作る。
- **問題提起**: 解決すべき課題を明確にする。
- **転換点**: 視点の転換、気づきを提供する。
- **解決策**: 具体的な解決策を提示する。
- **具体例**: 実例やデモで説得力を増す。
- **展望**: 将来像、次のステップを示す。
- **締め**: CTAまたは印象的なメッセージで締める。

---

## メモ

- このOutlineは「構造」なので、案件固有の素材は Input Bundle に寄せる
- 時間配分は想定尺に応じて調整する
- 各セクションは1スライド1メッセージの原則に従う
