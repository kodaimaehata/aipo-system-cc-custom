# Deck Review

## Review Axes

- chapter rhythm
- adjacency logic
- density rhythm
- repeated-layout fatigue

## Severity Levels

- `ok`:
  - no action needed
- `warning`:
  - recommendation is still usable, but a transition, density reset, or variation would improve the deck
- `fail`:
  - the current sequence breaks deck comprehension and should be changed before implementation

## Practical Rules

- Start with cover and agenda before dense evidence slides.
- Do not place dense graph/table slides back-to-back for long stretches.
- If comparison repeats, vary the emphasis target or insert a transition/orientation slide.
- Use low density for openers and transitions, medium for narrative slides, and controlled high density only for evidence.

## Continuity Notes To Return

- why this slide follows the previous one
- whether a transition slide is missing
- whether the deck overuses one pattern
- whether the current density is too high for its deck position

Each note should include one of `ok`, `warning`, or `fail`.
