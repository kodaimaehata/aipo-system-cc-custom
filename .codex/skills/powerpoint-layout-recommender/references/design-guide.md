# Design Guide

## Typography

- Font: `Noto Sans JP`
- Title: bold
- Body: regular
- Prefer short copy over shrinking type

## Palette

- white: `#FFFFFF`
- black: `#111111`
- secondary black: `#444444`
- red accent: `#C1121F`
- deep red: `#8E0E18`
- soft red tint: `#F7D9DC`

## Rules

- Stay inside the white/black/red family; vary tone, not hue.
- Use red only for emphasis or warning, not for all primary text.
- Keep slides clean and contrast-first.
- Dense evidence slides should still have a short lead that tells the audience what to read.
