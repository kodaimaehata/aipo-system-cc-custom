# /marp

**Usage**: `/marp [theme] [duration] [audience]`

Generates complete Marp slides from scratch (outline -> script -> slide).

## Inputs
- `theme`: Presentation topic/title (required)
- `duration`: Expected duration in minutes (default: 10)
- `audience`: Target audience (required)

1. Read `.claude/skills/make-marp-all/SKILL.md` and execute its instructions.
