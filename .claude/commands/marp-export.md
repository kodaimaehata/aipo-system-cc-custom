# /marp-export

**Usage**: `/marp-export [input_file] [--format pptx|pdf|html]`

Exports Marp Markdown to PPTX, PDF, or HTML using marp-cli.

## Inputs
- `input_file`: Path to Marp markdown file
- `--format`: Output format (default: pptx)

1. Read `.claude/skills/make-marp-export/SKILL.md` and execute its instructions.
