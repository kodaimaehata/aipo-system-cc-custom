# /draw-sequence

**Usage**: `/draw-sequence [description]`

Generates a draw.io sequence diagram from a text description.

## Inputs
- `description`: Message notation description of the sequence diagram (or provide inline)
- `output_path` (optional): Output file path (default: ./sequence.drawio)
