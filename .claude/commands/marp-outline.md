# /marp-outline

**Usage**: `/marp-outline [theme] [duration] [audience]`

Generates a presentation outline and input bundle for Marp slides.

## Inputs
- `theme`: Presentation topic/title (required)
- `duration`: Expected duration in minutes (default: 10)
- `audience`: Target audience (required)
