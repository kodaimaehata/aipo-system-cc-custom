# /deliver

**Usage**: `/deliver [TaskID] [LayerPath]`

Loads the **AIPO Deliver Skill** to execute a task.

## Inputs
- Optional: task id (e.g., `T001`)
- Optional: layer path (recommended when multiple layers exist), e.g. `programs/<project_name>` or `programs/<project_name>/sublayers/<sublayer_name>`

## Execution
1. Read `.claude/skills/aipo-core/SKILL.md` and follow its rules.
2. Read `.claude/skills/aipo-deliver/SKILL.md` and execute its instructions.
