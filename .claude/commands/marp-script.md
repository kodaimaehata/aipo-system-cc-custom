# /marp-script

**Usage**: `/marp-script [outline_path] [input_bundle_path]`

Generates a presentation script from outline and input bundle.

## Inputs
- `outline_path`: Path to outline file (or provide inline)
- `input_bundle_path`: Path to input bundle file (or provide inline)
