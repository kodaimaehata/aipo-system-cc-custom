# /draw-flowchart

**Usage**: `/draw-flowchart [description]`

Generates a draw.io flowchart from a text description.

## Inputs
- `description`: Natural language description of the flowchart (or provide inline)
- `output_path` (optional): Output file path (default: ./flowchart.drawio)
